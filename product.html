Windows Configuration

WinConfig is based on my personal research on several topics, which I began documenting around July 2024.

Lifetime license 9,99 EUR

Includes updates & Discord role.

Powered by

Noteable Windows Configuration Tool Features:

  • Transparent execution logs so users can review every action, these include the value, data, type and if given the amount of modified keys. Specific options will show the PS code that got executed.
  • Dynamic state detection instead of relying on cached registry values, this includes static/dynamic (pattern) and all used cmdlets (which are also saved in the registry but on uncommon ways). Note that this feature affects app performance, as several commands are called every time you switch trough sections. However, you always see the current state, which more than compensates for the resulting wait time.
  • Each option is backed up by documentation, instead of just general "optimize" prompts with an useless text.
  • Guidance includes lesser known data alongside common adjustments, for futher customization for experienced users.
  • Every control is available individually instead of being merged into large bundles.
  • Supports dark/light modes, custom color schemes, and layout preferences for search filters, rows, columns, and panel widths.

Purchase, privacy, and license notes

Data Privacy

WinConfig only references hardware identifiers required for license verification: UUID, CPU ID, and board ID. If any are unavailable, it can fall back to disk serial, MAC address, video controller ID, sound device ID, or USB controller ID. No additional system data is collected, transmitted, or stored. Learn more about the firmware identifiers via the official SMBIOS documentation.
