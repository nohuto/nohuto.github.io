Security & Privacy

Privacy statement

This site is a personal website for showing off my projects and product. It doesn't use analytics, trackers, or cookies. No account system exists and no user data is stored by the site itself.

What is collected

The hosting stack (Cloudflare and GitHub Pages) may log standard request metadata such as IP address and user agent for operational or abuse prevention reasons.

Third party requests

Some pages include media hosted on GitHub (raw.githubusercontent.com). Those resources are loaded directly from GitHub and will disclose basic request metadata to GitHub. External links to GitHub, YouTube, Discord, and PayPal take you to those services when clicked.

Repo descriptions

Project descriptions are served from a local JSON file generated at build time. The site does not call the GitHub API from your browser.

Browser policies

The site sets a Content Security Policy (CSP), a strict Referrer Policy, and a restrictive Permissions Policy. On GitHub Pages these are applied via meta tags, and can also be enforced at the Cloudflare edge for stronger coverage. These reduce script injection, referrer leakage, and unnecessary browser features.
