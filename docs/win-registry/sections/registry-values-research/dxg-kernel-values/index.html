DXG Kernel Values

These are default values I found in dxgkrnl.sys, see assets/dxg-values for pseudocode snippets I used / records/Graphics-Drivers.txt for all values that get read on boot.

The GraphicsDrivers\Scheduler / GraphicsDrivers\MemoryManager values are from dxgmms2.sys, I used the drivers from 23H2/25H2 since they differ at some point. See dxgmms2 for all used files.
